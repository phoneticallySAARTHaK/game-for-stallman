A game for Stallman

Since Mr. Stallman does not use javascript in his browser, all the web-based games (and most of the websites) must be inaccessible to him.

Being a follower of Libre software philosophy, I developed a game which doesn't require javascript to play.

Hope you enjoy it, fellow GNU/Linux users!
(Scroll down for rules)

Rules

  • Since there is no javascript used, the "randomisation" factor is you.
  • More precisley, CSS animations and you.
  • The moving box and your user action stimulates a random choice. You then directed to the game board. (try clicking in different parts of the box, at different times).
  • You have to find the FSF Logo, hidden at a random position in the grid.
  • Without looking at the source code, you have to estimate the transformation, and find the number, before the time runs out!
